# Amazon Prime Video Analytics Dashboard

## Project Overview

This project presents an interactive **Amazon Prime Video Analytics Dashboard** built using **Power BI**. The dashboard provides a comprehensive analysis of Amazon Prime Video's content library from **1920 to 2021**, enabling users to explore trends in movies, TV shows, ratings, genres, directors, release years, and global content distribution.
The goal of this project is to demonstrate data visualization, dashboard design, and business intelligence skills by transforming raw entertainment data into meaningful insights.

## Key Insights

### Content Type Distribution
- Movies account for approximately **80%** of the platform's content.
- TV Shows contribute around **20%**.

### Growth Trend
- Significant growth observed after **2010**.
- Content additions accelerated rapidly between **2015–2021**.

### Popular Genres
- Drama
- Comedy
- Documentary
- Kids & Family
- Action & Drama

### Global Reach
- Content is distributed across multiple countries, showcasing Amazon Prime Video's international presence.
